Samenvatting van het college

In het 5de college stond Introductie HTML & CSS centraal. We begonnen het college zoals gewoonlijk met een korte terugblik op de week ervoor. We gingen daarna de HTML theorie bespreken wat we voor het college moesten doen en we gingen daarna een Kahoot doen over de HTML theorie. Daarna kregen we wat tips over HTML en toen mochten we zelf aan de slag ermee.

We kregen genoeg tijd om zelf ermee aan de slag te gaan door zelf de opbouw van de website te bouwen.

Daarna gingen we verder met de introductie van CSS, we kregen hierin een korte demo over hoe CSS werkte.

Als laatst kregen we een korte introductie over JavaScript met een korte demo en er werd ons verteld dat JavaScript niet een vereiste was voor onze website, maar je mag het altijd toevoegen als het voor jou iets toevoegt.


Wat is HTML?

HTML staat voor Hyper Text Markup Language. HTML is de standaard ‘taal’ voor het bouwen van webpagina’s, HTML geeft het structuur aan van de website. Het bestaat doormiddel van verschillende elementen en die elementen geven weer aan je browser door hoe dat element moet worden weergeven.

De volgende afbeelding geeft een simpele structuur/indeling van een website aan in HTML taal.


Wat is CSS?

CSS staat voor Cascading Style Sheets. CSS regelt de opmaak van de HTML elementen. Ook al kun je opmaak toevoegen in HTML zelf, wordt er vaak gekozen om het via CSS te regelen omdat het makkelijker gaat en je code in HTML niet zo onoverzichtelijk wordt. HTML is meer om de content van een webpagina te weergeven en CSS is echt gericht op de opmaak van de HTML elementen.
