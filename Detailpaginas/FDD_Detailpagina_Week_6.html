Samenvatting van het college

Dit 6de college stond het onderwerp CSS Frameworks centraal. Gewoonlijk begonnen we met een korte herhaling van de week ervoor. Daarna kregen we feedback op de leertaken voor het college, dat was het doornemen van het CSS theorie en een quiz maken.

We kregen toen korte demo’s over hoe we bepaalde dingen konden bereiken met CSS en kregen we even de tijd om aan onze website te werken.

Daarna gingen we verder met het onderwerp CSS Frameworks, dat is een soort van online bibliotheek waar je verschillende gecodeerde CSS bestanden/code kunt vinden en kunt toepassen op je eigen website of zelf een kunt aanpassen en dan toepassen.

We werden toen geïntroduceerd in een accessibility tool die ingebouwd zit in de meeste browsers en moesten we als voorbeeld een test uitvoeren met die tool op de CHE website. Daaruit kwam een rapport met een score over de accessibility van de website en de verbeterpunten. We kregen toen nog wat tips om dit soort testen uit te voeren op onze eigen website om ook de accessibility te controleren.


Wat is CSS Framworks?

Simpel gezegd is CSS Frameworks een online bibliotheek vol met verschillende CSS code. Met de CSS code kun je inspiratie opdoen of toepassen op je eigen website. Er zijn verschillende CSS Frameworks bibliotheken waaruit je kunt kiezen.
