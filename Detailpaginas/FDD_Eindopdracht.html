Week 4 | Design en Accessibility

Voor het 4de college was het de bedoeling dat we een moodboard gingen maken. In dit moodboard bevat lettertypes, kleuren en websites die jou inspireren of aanspreken. Voor dit college moesten we ook een hi-fi wireframe gaan maken. Dit hi-fi wireframe maakten we met het online programma Figma, deze wireframe moest de homepagina, contact pagina en een detailpagina bevatten die zijn uitgewerkt in een hi-fi wireframe. Ik heb alleen de homepagina kunnen maken en ben daarna gestopt met het gebruik van dit programma omdat ik niet meer tijd in het Hi-Fi wireframe wou stoppen.

Week 5 | Introductie HTML & CSS

Voor het 5de college moesten we theorie doornemen van HTML, met de doorgenomen theorie moesten we een kennistoets gaan maken.

We gingen met het programma Visual Studio Code aan de slag met het schrijven van de HTML code voor de website, we moesten ook wat extensies toevoegen dat het proces wat makkelijker maakt. Tijdens het college kregen we ook de tijd om onze eerste webpagina te gaan maken met HTML.

We hebben ook het advies gekregen om een W3C check uit te voeren op onze eigen website. De W3C check controleert je geschreven code op fouten.

Zoals u in de afbeeldingen hierboven ziet geeft de W3C check fouten aan op mijn HTML code maar deze fouten kunt u negeren omdat ik mijn code op een bijzondere manier heb opgemaakt (extra opmaak extensie in Visual Studio Code) daardoor ziet de W3C check dit als een fout. Als u kijkt bij mijn CSS code dan ziet u dat daarin geen fouten bevatten.

In dit college werd ook de vraag gesteld of je de website bouwt met het gedachte voor de desktop eerst of voor de mobiel eerst. Dit houdt in dat je de website optimaliseert voor desktop gebruik of gebruik voor op de mobiel. Ik heb voor de desktop gekozen want ik vond dat handiger, dit zie je ook terug als je mijn website in een mobiele omgeving toont. In de afbeeldingen hieronder kun je wat verschillen zien zoals dat het menubalk onderelkaar zit en dat je in het menubalk moet scrollen.


Week 6 | CSS Frameworks

De theorie van CSS moesten we voor het 6de college doornemen. Met deze doorgenomen theorie moesten we ook hierbij een kennistoets maken alleen dit was ook nog in combinatie met wat theorie nog over HTML.

We kregen ook in het college een introductie over een accessibility tool, deze tool zat ingebouwd in je browser en kan je gebruiken om een accessibility score te genereren. Ik heb deze tool ook uitgevoerd op de homepagina van mijn website en in de afbeelding hieronder kunt u de score zien. De score laat zien dat mijn homepagina geen 100 scoort dat vanwege het contrast ratio, dit was ook eerder besproken als u naar boven scrolt. Het tweede is dat sommige links niet goed zijn gedefinieerd en heel algemeen zijn beschreven.


Week 7 | DNS & Uploaden van je site

Met het 7de en ook het laatste college werd de nadruk gelegd op het maken van de website. Dit college gaf ook uitleg hoe je je eigen website op het internet kan krijgen. Zoals u kunt zien staat mijn website succesvol op het internet want anders zou u hier niet zijn en dit kunnen lezen.
Het online zetten van de website ging met het programma CloudBerry. Het is een simpel proces waarbij je de website bestanden op de server zet.
